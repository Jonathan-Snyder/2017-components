Welcome to the new templates!

Feel free to check back here for updates. I'll upload the latest code every time we get a pull request on the github repository (https://github.com/byuweb/2017-components).

As of right now, you need to include the following to use the components:

  • The web component polyfills: <script src="https://lscdn.byu.edu../../bower_components/webcomponentsjs/webcomponents.js"></script>
  • The script file for the components: <script src="https://lscdn.byu.edu../../dist/components.js"></script>
  • The font-awesome stylesheet: <link rel="stylesheet" href="https://lscdn.byu.edu../../bower_components/font-awesome/css/font-awesome.min.css">
  • The default site.css: <link rel="stylesheet" href="https://lscdn.byu.edu../../css/site.css">
  • The Hoeffler Fonts Stylesheet: <link type="text/css" rel="stylesheet" href="//cloud.typography.com/75214/6517752/css/fonts.css" media="all" />
